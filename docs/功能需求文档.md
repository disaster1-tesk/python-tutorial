# Python 教程 Web 平台 - 功能需求文档

## 项目概述

**项目名称**: Python 教程学习平台
**项目类型**: Web 应用（在线学习系统）
**核心功能**: 24 个 Python 知识模块的在线学习、练习、测验系统

## 目标用户

- Python 初学者
- 需要系统学习 Python 的开发者
- 想要巩固 Python 知识的学习者

## 功能需求

### 1. 首页

- 显示学习进度统计（总模块数、已完成、进行中、完成率）
- 核心功能介绍卡片
- 学习路径建议（按分类展示模块）
- 快速开始按钮

### 2. 知识学习模块

- **24 个知识模块**，包括：
  - 基础阶段：base_syntax, data_types, control_flow, functions, classes_objects, exception_handling, file_operations, modules_packages
  - 进阶阶段：advanced_features, type_hints, concurrency, standard_library, third_party_libraries
  - 应用阶段：database, design_patterns, networking, data_processing, web_development
  - AI/ML：ai_ml_basics, deep_learning, nlp_and_llm, cv_computer_vision
  - 工程化：unit_testing, debugging

- 每个模块包含：
  - 知识点讲解（description.md）
  - 代码示例（example.py）
  - 学习状态追踪

- 页面功能：
  - Tab 切换（知识点 / 代码示例）
  - 标记已完成
  - 侧边栏快速导航

### 3. 练习题系统

- 支持按模块筛选
- 显示难度（简单/中等/困难）
- 显示分值和标签
- 题目描述和起始代码

### 4. 章节测验

- 支持按模块筛选
- 选择题形式
- 显示答案和解析
- 覆盖 13+ 个模块

### 5. 学习统计

- 模块完成进度
- 练习题完成情况
- 测验成绩记录

### 6. AI 智能助手

- 悬浮框形式（固定右下角）
- 支持多 AI 提供商：
  - Ollama（本地）
  - OpenAI
  - Claude
- 实时对话功能
- 配置面板（API 地址、Key、模型）

## 技术要求

### 框架选择

- **推荐**: Flask + 原生 HTML/CSS/JS
- 端口: 8502
- 轻量级、易于维护

### 数据存储

- JSON 文件存储进度数据
- 模块数据从文件系统读取（与教程目录同步）

### 响应式设计

- 支持桌面端和移动端
- 现代化的 UI 设计

## 验收标准

1. ✅ 首页正常显示，显示统计数据
2. ✅ 24 个模块都可以访问和学习
3. ✅ 练习题可以按模块筛选显示
4. ✅ 测验题可以按模块筛选显示
5. ✅ 学习进度可以保存
6. ✅ AI 悬浮框可以在所有页面显示
7. ✅ 响应式布局正常
