## 技术实现细节

### 目录结构
```
web/
├── app.py                 # Flask 主应用（单一文件）
├── config.py             # 配置（已存在）
├── requirements.txt      # 依赖（已存在）
├── data/                 # 数据目录（已存在）
│   ├── progress.json
│   ├── exercises_v2.json
│   └── quizzes.json
├── templates/            # HTML 模板
│   ├── base.html         # 基础模板（含 AI 悬浮框）
│   ├── index.html        # 首页
│   ├── learn.html        # 学习页
│   ├── exercises.html    # 练习题
│   ├── quiz.html         # 测验
│   └── stats.html        # 统计
└── static/
    └── css/
        └── styles.css    # 样式文件
```

- 主应用：`http://localhost:8502`
